# estudio_musical
Sisteminha web para o gerenciamento de bandas em um estúdio musical. Construído em JavaEE (JSP, Servlets, JSTL, Tom Cat, etc), Bootstrap, Mysql e JQuery. O padrão MVC também foi utilizado.
